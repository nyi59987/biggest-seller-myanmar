DEPLOY INSTRUCTIONS

Upload the CONTENTS of this folder, or upload this ZIP as a static site.
The file index.html must be at the website root.

Included files:
- index.html: main website page
- 404.html: fallback page for hosts like GitHub Pages
- _redirects: fallback rule for Netlify
- vercel.json: rewrite rule for Vercel

If you still see 404:
1. Make sure your hosting provider's publish/root directory is the folder that contains index.html.
2. Do not deploy only the old file named "Biggest Seller Myanmar(1).html" unless your host lets you set that as the homepage.
3. Clear the deployment and upload this ZIP again.
